When I started this blog I knew I wanted to have some way to organize and keep track of interesting material I came across concerning web application development.  This page is my attempt at congregating all of that material in one place both for my own benefit and anyone else who might be interested

Learning Ruby on Rails

  • Rails Guides - rubyonrails.org - Rails guides by the Rails core team
  • Ruby on Rails Tutorial by Michael Hartl - An excellent tutorial working through the basics of Rails in an easily digestible format
  • I Want To Learn Ruby - A comprehensive directory of sites helping teach Ruby and Rails
  • Practicing Ruby - An excellent compilation of essays related to various components of Ruby and web application programming in general
  • Git The Simple Guide - Simple breakdown of Git workflow and commands
  • W3 Schools - CSS and HTML references and tutorials

Gems

  • Ruby Toolbox - A great site to easily find a gem for a specific task.  Rates and compares against other similar gems so you know what is best for your app

Ruby Podcasts

  • RailsCasts - Screencasts covering a vast array of topics ranging from individual gems to meta programming techniques
  • Ruby5 -  ~5 minute podcasts covering "the latest news in the Ruby and Rails community"
  • Ruby Rogues - More discussion based podcast covering interesting topics and developments in the Ruby community
  • 5by5 Ruby on Rails Podcast

Programming Challenge Sites

  • Ruby Quiz - Problems to be solved with Ruby
  • Project Euler - Mathematical problems ideally solved through programming (not Ruby specific)
  • Code Quizzes - More traditional quizzes and questions than challenges per say

Design

  • Bootstrap: Getting Started - Guides from the official Bootstrap site
  • Tealeaf blog: Rails and Bootstrap - I found this blog series incredibly helpful when I was learning how to set-up and use Bootstrap
  • Pixlr - An image editor in my browser anytime I need it?  Aww yeah!
  • Balsamiq - Mockup software to make prototyping easier

Virtual Machines and Vagrant

  • JumpStartLab: Vagrant with Ruby on Rails - This was the most helpful resource when I was trying to set up my Vagrant box and provision it with all the necessary components for a Ruby on Rails stack
  • Vagrant Docs - Official documentation and guides from the Vagrant website

Startups

  • TechCrunch - News and articles for startups and development

Miscellaneous

  • Reddit: Webdev - Subreddit for web developers
  • 40 Famous Websites Built with Ruby on Rails
  • Rubular - A Ruby regular expression editor
  • You're never too old... - Infographic on successful entrepreneurs who found success later in life
